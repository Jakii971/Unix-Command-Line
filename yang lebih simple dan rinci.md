## Dokumentasikan tiap-tiap langkah

```
1. mkdir Jaki
2. cd Jaki
3. mkdir sekolah kerja
4. cd sekolah
5. touch ijazah.txt
6. vim ijazah.txt (mengisi teks)
7. cat ijazah.txt
8. touch portofolio.txt
9. vim portofolio.txt (mengisi teks)
10. cd ..
11. cd kerja
12. touch cv.txt
13. vim cv.txt
14. cat cv.txt
15. mv portofolio.txt ../kerja
```
